Introduction

Python is a popular general-purpose programming language that can be used for a wide range of applications. Some key features of Python include:

  • High-level and interpreted language
  • Dynamically typed
  • Automatic memory management
  • Supports multiple programming paradigms (object-oriented, procedural, functional, etc.)
  • Large standard library and extensive third party modules
  • Cross-platform compatibility

This document provides a basic overview of Python syntax, variables, data types, and flow control statements to help you get started with Python programming.

What you should already know

Before starting with Python, you should have a basic understanding of:

  • Programming concepts like variables, loops, conditional statements, functions, etc.
  • Some basic math like arithmetic, algebra, etc.
  • Basic computer and command line usage

If you have experience with other programming languages like C, Java, JavaScript, etc., that will help you pick up Python faster.

Hello World

Let's start with the traditional "Hello World" program:

                
                  print("Hello World!")

This prints the text "Hello World!" to the output.

To run this, save it as hello.py and execute it:

                
                    $ python hello.py
                    Hello World!
Variables

Variables are used to store values in Python. They can be assigned with the = operator:

                
                    name = "John"
                    age = 25

You can print variables like:

                
                    print(name)
                    print(age)

Variables don't need type declarations in Python. The type is determined automatically based on the value assigned.

Declaring Variables

Python has no command for declaring variables. A variable is created the moment you assign a value to it:

                
                    x = 5
                    y = "Hello"

Variables must start with a letter or underscore. They can contain letters, numbers and underscores.

Variable Scope

The scope of a variable is the part of the code where it is visible.

Variables defined inside a function are local to that function. They cannot be accessed outside:

                
                    def myFunction():
                        x = 5
  
                    myFunction()
                    print(x) #Error, x not defined

Variables defined outside functions are global. They can be accessed globally:

                
                    x = 5 

                    def myFunction():
                    print(x)

                    myFunction() #Prints 5
                    print(x) #Prints 5
Global Variables

The global keyword declares a variable global inside a function. This gives the function access to change the global variable:

                
                    x = 5

                    def myFunction():
                    global x
                    x = 7

                    myFunction()  
                    print(x) #Prints 7
Constants

Constants are variables whose value stays the same throughout the program. By convention, constants are usually named in all uppercase letters.

To define a constant, use the final keyword:

                
                    PI = 3.14
                    GRAVITY = 9.8 

                    PI = 5 #Error, cannot change value
Data Types

Python has several built-in data types:

  • Numbers (int, float, complex)
  • Strings
  • Booleans
  • List
  • Tuple
  • Set
  • Dictionary

You can check data types with type():

                
                    x = 5
                        print(type(x)) #Prints 

                        y = 5.5
                        print(type(y)) #Prints
if...else Statement

The if statement executes code if a condition is true. The else statement executes code if the condition is false:

                
                    age = 20
                    if age >= 18:
                        print("You are an adult")
                    else:
                        print("You are a minor")

Multiple if...elif...else statements can be chained:

                
                    if age <= 12:
                    print("Child")
                  elif age > 12 and age < 18:
                    print("Teenager")  
                  else:
                    print("Adult")
while Loop

The while loop runs as long as a condition is true:

                
                    count = 0
                    while count < 5:
                    print(count)
                    count += 1

The break keyword exits the loop, and continue skips to the next iteration.

Function Declarations

Functions are declared with the def keyword:

                
                    def myFunction(param1, param2):
                        print(param1 + param2)
                    
                    myFunction(5, 3) #Prints 8

Parameters are optional, and can have default values:

                
                    def greeting(name="User"):
                        print("Hello " + name)

                    greeting("John") #Hello John
                    greeting() #Hello User
